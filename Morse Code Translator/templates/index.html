Notes & Rules

Important rules for using the Morse Code Translator:

  • Supported Characters: Letters (A-Z), Numbers (0-9), and basic punctuation: . , ? / - ( )
  • Case Insensitive: Both uppercase and lowercase letters work the same way
  • Word Separation: Spaces between words are represented as / in Morse code
  • Letter Separation: Individual Morse letters are separated by single spaces
  • Format Example:
    "HELLO WORLD" becomes .... . .-.. .-.. --- / .-- --- .-. .-.. -..
  • Invalid Characters: Special symbols like @, #, $, %, etc. are not supported and will show an error
  • Keyboard Shortcut: Press Ctrl + Enter to convert text quickly
